**A**: It's now a collapsible menu! Click the "Batch Drafts" text or arrow to expand it.

### Q: What are those yellow "New" badges?

## ğŸ› Troubleshooting

### Widget Not Showing
1. Confirm you're logged in as **Chairperson**
2. Check if **academic period is selected** (top-right dropdown)
3. Try refreshing the page (Ctrl+R / Cmd+R)
4. Clear browser cache (Ctrl+Shift+Delete)

### Sidebar Menu Won't Expand
1. Check if JavaScript is enabled
2. Try a different browser
3. Disable browser extensions temporarily
4. Check console for errors (F12)

### Statistics Show Zero
1. Verify you have batch drafts created
2. Check academic period matches your batches
3. Confirm course assignment is correct
4. Contact support if issue persists

### Buttons Don't Work
1. Check if you have proper permissions
2. Verify routes are cached (`php artisan route:cache`)
3. Check for JavaScript errors (F12 console)
4. Try logging out and back in
